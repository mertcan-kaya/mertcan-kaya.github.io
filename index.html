Selected Publications & Core Formulations

📐 View Mathematical Formulation: Distractor-Aligned Variance (DAV)

To isolate true physical human motor interference metrics from intentional task trajectories, the framework maps tracking path deviations orthogonal to the core geometric task vector \(\mathbf{x}(t)\):

$$\text{DAV} = \frac{1}{T} \int_{0}^{T} \left\| \mathbf{e}(t) - \left( \mathbf{e}(t)^T \hat{\mathbf{v}}_d(t) \right) \hat{\mathbf{v}}_d(t) \right\|^2 dt$$
  • Journal Article • Interaction Studies • 2026
    Motor interference of elbow configuration changes in human-robot interaction
    Problem: Collaborative robots changing kinematic configurations in shared sightlines provoke hidden cognitive friction and sub-optimal motion path corrections in humans.
    Contribution: Tracked visual gaze and upper-body multi-joint trajectories to chart human biomechanical path adaptation relative to robot null-space positioning.
    Outcome: Verified that null-space re-orientation drives involuntary trace adaptations, revealing clear boundaries for proactive trajectory layout.
  • Journal Article • Journal of Intelligent & Robotic Systems • 2024
    A novel recursive algorithm for the implementation of adaptive robot controllers
    Problem: Traditional adaptive controllers demand high-dimensional regressor structural formulations, capping loop iteration cycles on industrial embedded processors.
    Contribution: Developed an alternative recursive inversion mapping framework leveraging multi-body inverse Newton-Euler dynamics parameters.
    Outcome: Eliminated explicit matrix build parameters completely, cutting loop overhead and tracking lag while expanding tracking speeds on running physical processors.
