Hardware Projects & Systems

Detailed breakdowns of physical robot deployments, tracing the engineering stack from low-level embedded microcontrollers up to high-level ROS 2 behavior trees.

1. Autonomous Mobile Navigation & SLAM

Objective

Architecting robust, real-time spatial occupancy grids and dynamic local/global costmaps for reactive trajectory adaptation in shared environments. This project verifies the integration of sensor arrays to allow mobile bases to negotiate dynamic obstacles safely.

Technical Specifications

  • Hardware: TurtleBot 4, AgileX Scout Mini, 2D/3D LiDAR, OAK-D Spatial Cameras.
  • Software Stack: ROS 2 (Humble/Iron), Nav2 Framework, Cartographer SLAM, Python, C++.
  • Control Logic: Behavior Trees (BT.CPP), Extended Kalman Filters (EKF), AMCL.

2. Compliant Teleoperation & Physical HRI

Objective

Implementing hybrid position/force control loops and telemanipulation frameworks. The goal is to ensure stable industrial pick-and-place trajectories while allowing continuous, safe kinetic adaptation during direct human physical contact.

Technical Specifications

  • Hardware: Franka Emika Panda (7-DoF), Universal Robots (UR3).
  • Software Stack: ROS Noetic, MoveIt Motion Planning Framework, libfranka.
  • Control Logic: Impedance Control, Hybrid Position/Force Control, Inverse Kinematics.

3. Real-Time Predictive Human Tracking

Objective

Empirical verification of predictive manipulator tracking loops. The system executes seamless task-space path updates to follow human spatial vectors actively in real time, serving as the foundational layer for evaluating motor interference and anthropomorphism.

Technical Specifications

  • Hardware: Pepper, NAO Systems, High-Fidelity RGB-D Vision Systems.
  • Software Stack: ROS, OpenCV, Python, C++.
  • Control Logic: Visual Servoing, Trajectory Optimization, Human Intent Prediction.

4. Embedded Actuation & Microcontroller Firmware

Objective

Deployment of modular, low-latency device abstraction layers designed to link active high-level ROS execution graphs down to custom hardware micro-actuators. This validates end-to-end full-stack robotics competence.

Technical Specifications

  • Hardware: Raspberry Pi Pico, Custom Chassis, Multi-Axis Servo Arrays, Synchronous Scissor-Lift.
  • Software Stack: Micro-ROS, FreeRTOS, C/C++, Serial Communication (UART/I2C/SPI).
  • Control Logic: PID Motor Control, Kinematic Chain Solvers, Pulse-Width Modulation (PWM).
